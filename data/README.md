# Data

This repository does not include the full MVTec AD dataset.

Please obtain the dataset from the official MVTec AD source and place it under:

```text
data/mvtec_ad/
```

Official dataset paper:
- Paul Bergmann, Kilian Batzner, Michael Fauser, David Sattlegger, and Carsten Steger, "The MVTec Anomaly Detection Dataset: A Comprehensive Real-World Dataset for Unsupervised Anomaly Detection," IJCV 2021.

Official dataset page:
- https://www.mvtec.com/company/research/datasets/mvtec-ad

No dataset files are bundled in this public release.
