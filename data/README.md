# Data

This repository does not include the full MVTec AD dataset.

Please obtain the dataset from the official MVTec AD source and place it under:

```text
data/mvtec_ad/
```

Official dataset paper (APA):
- Bergmann, P., Batzner, K., Fauser, M., Sattlegger, D., & Steger, C. (2021). The MVTec anomaly detection dataset: a comprehensive real-world dataset for unsupervised anomaly detection. *International Journal of Computer Vision*, 129(4), 1038-1059.

Official dataset page:
- https://www.mvtec.com/company/research/datasets/mvtec-ad

Selected papers using this dataset (APA):

- Roth, K., Pemula, L., Zepeda, J., Schölkopf, B., Brox, T., & Gehler, P. (2022). Towards total recall in industrial anomaly detection. In *Proceedings of the IEEE/CVF conference on computer vision and pattern recognition* (pp. 14318-14328).
- Defard, T., Setkov, A., Loesch, A., & Audigier, R. (2021, January). Padim: a patch distribution modeling framework for anomaly detection and localization. In *International conference on pattern recognition* (pp. 475-489). Cham: Springer International Publishing.

No dataset files are bundled in this public release.
